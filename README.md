# Настройка бота

## 1. Установить зависимости
```
pip install -r requirements.txt
```

## 2. Узнать свой chat_id
1. Напишите боту @FacebookApartanent_bot любое сообщение, например "старт".
2. Откройте в браузере (замените TOKEN на свой токен):
   https://api.telegram.org/botTOKEN/getUpdates
3. В ответе найдите число вида `"chat":{"id":123456789` — это ваш chat_id.

## 3. Получить API-ключ Claude
Зайдите на https://console.anthropic.com → раздел API Keys → создать ключ.

## 4. Указать настройки
Перед запуском задайте переменные окружения:
```
export TELEGRAM_BOT_TOKEN="8837144583:AAEScqV_UUtDtvObQ6syDWRxHud5eft6mFA"
export TELEGRAM_CHAT_ID="ваш_chat_id"
export ANTHROPIC_API_KEY="ваш_ключ"
```

## 5. Проверить вручную
```
python listing_bot.py
```
Если всё настроено верно, в Telegram придёт пост с новым объявлением (или будет написано "Найдено новых объявлений: 0", если всё уже отправлено).

## 6. Сделать автоматический запуск по расписанию
Добавьте в cron (например, проверка каждый час):
```
0 * * * * cd /путь/к/папке/со/скриптом && /usr/bin/python3 listing_bot.py >> log.txt 2>&1
```

## Важно про безопасность
Токен бота был один раз отправлен в переписке — после первого успешного теста рекомендую перегенерировать его через @BotFather командой `/revoke`, и вписать новый токен везде, где он использовался.
